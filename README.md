# candidatapp
A desktop application for managing job applicants, tracking open positions and interviews, using WPF and MVVM with responsive UI and efficient data binding.
 - Candidate management (personal details, experience, role)
 - Position tracking (open roles, descriptions, status)
 - Application workflow (applied date, source, status updates)
 - Interview scheduling (linked to candidates and applications)
   
⚠️  This project is currently **under development** (features incomplete/in progress)
